# 渐进式超负荷原则

渐进式超负荷是肌肉生长和力量提升的核心原则。

## 什么是渐进式超负荷

渐进式超负荷指逐渐增加训练负荷，持续给肌肉新的刺激。

没有新的刺激，身体就不会产生适应和改变。

## 增加负荷的方式

### 1. 增加重量
最直接的方式
- 例如：深蹲从60kg增加到65kg
- 每周增加2.5-5%

### 2. 增加次数
在相同重量下增加次数
- 例如：60kg从8次增加到10次

### 3. 增加组数
增加训练量
- 例如：从3组增加到4组

### 4. 缩短休息
增加密度
- 例如：休息从90秒减到60秒

### 5. 改善动作质量
更严格的动作标准

## 实施方法

### 双重进阶法
1. 先在相同重量下增加次数
2. 达到目标次数后增加重量
3. 重复循环

例如：
- 第一周：60kg x 8次
- 第二周：60kg x 9次
- 第三周：60kg x 10次
- 第四周：62.5kg x 8次

### 记录训练日志
- 记录每个动作的重量、组数、次数
- 对比上周数据
- 尝试每次都有进步

## 注意事项

1. 不要急于求成
2. 保持动作标准优先
3. 充足恢复
4. 周期性调整
5. 避免过度训练

## 进步速度

### 新手期
- 几乎每次都能增加
- 进步最快

### 中级期
- 每周或每两周增加
- 进步变慢

### 高级期
- 每月或更长时间增加
- 需要更精细的计划
