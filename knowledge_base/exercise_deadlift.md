# 硬拉动作指南

硬拉是最有效的全身力量训练动作之一，主要锻炼后链肌群。

## 动作类型

### 传统硬拉
- 双脚与髋同宽
- 手臂在膝盖外侧
- 适合大多数人

### 相扑硬拉
- 双脚宽于肩
- 手臂在膝盖内侧
- 适合腿长或髋关节灵活者

## 动作要领

### 起始姿势
1. 杠铃位于脚掌中部上方
2. 双脚与髋同宽，脚尖略外展
3. 臀部下沉，膝盖弯曲
4. 握住杠铃，手臂伸直
5. 肩胛骨在杠铃正上方
6. 背部挺直，胸部挺起

### 拉起阶段
1. 脚跟用力蹬地
2. 臀部和膝盖同时伸展
3. 保持背部挺直
4. 杠铃贴近身体
5. 站直后锁定髋部

### 下放阶段
1. 先屈髋，后屈膝
2. 控制速度下放
3. 保持背部挺直

## 常见错误

1. **弯腰拉起**：极易导致腰椎受伤
2. **臀部先抬起**：变成下背部发力
3. **重心前移**：脚跟离地
4. **过度伸展**：锁定后不应后仰

## 训练建议

- 初学者必须从轻重量开始
- 使用正反握或助力带防止脱手
- 每周1-2次，每次3-5组，每组3-8次
