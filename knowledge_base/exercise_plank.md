# 平板支撑指南

平板支撑是最基础的核心训练动作，锻炼腹横肌、腹直肌等核心肌群。

## 动作要领

### 标准平板支撑
1. 俯卧，前臂撑地
2. 肘部在肩膀正下方
3. 双脚并拢，脚尖着地
4. 身体从头到脚成一条直线
5. 收紧腹部和臀部
6. 保持头部自然，目视地面

### 高位平板支撑
- 双手撑地代替前臂
- 更适合初学者

## 常见错误

1. **塌腰**：核心力量不足，可先做跪姿
2. **撅臀**：臀部过高，减少核心参与
3. **耸肩**：保持肩胛骨稳定
4. **憋气**：应保持正常呼吸

## 进阶变式

1. **侧平板**：锻炼腹斜肌
2. **动态平板**：交替抬手或抬脚
3. **负重平板**：背部放置重量
4. **健身球平板**：增加不稳定因素

## 训练建议

- 初学者：每组20-30秒
- 进阶者：每组60秒以上
- 每次3-4组
- 每周3-4次

## 注意事项

- 保持呼吸，不要憋气
- 如果腰痛，立即停止并检查姿势
- 可以每天练习
