# 深蹲动作指南

深蹲是力量训练中最基础也是最重要的复合动作之一，被称为"动作之王"。它主要锻炼大腿、臀部和核心肌群。

## 动作要领

### 起始姿势
1. 双脚与肩同宽或略宽于肩，脚尖微微外展（约15-30度）
2. 挺胸收腹，目视前方
3. 肩胛骨后缩下沉，保持脊柱中立位

### 下蹲阶段
1. 臀部先向后坐，像要坐椅子一样
2. 膝盖沿脚尖方向弯曲，不要内扣
3. 下蹲至大腿与地面平行或略低
4. 保持重心在脚掌中部，脚跟不离地

### 起身阶段
1. 臀部和大腿发力，脚跟用力蹬地
2. 保持躯干稳定，不要弯腰
3. 呼气同时站起，回到起始姿势

## 常见错误

1. **膝盖内扣**：这是最常见的错误，容易导致膝盖受伤。应该保持膝盖与脚尖方向一致。

2. **弯腰驼背**：核心力量不足时容易出现。需要加强核心训练，保持脊柱中立。

3. **脚跟离地**：说明重心前移，可能是踝关节活动度不足。可以先做半蹲练习。

4. **下蹲不够深**：建议至少蹲到大腿与地面平行，以充分激活臀部肌肉。

## 训练建议

- 初学者建议从徒手深蹲开始，掌握正确姿势后再负重
- 每周训练2-3次，每次3-4组，每组8-12次
- 进阶后可尝试高脚杯深蹲、杠铃深蹲等变式

## 肌肉参与

主要肌群：
- 股四头肌（大腿前侧）
- 臀大肌
- 腘绳肌（大腿后侧）

辅助肌群：
- 竖脊肌
- 腹直肌
- 小腿肌群
