# 卧推动作指南

卧推是发展上肢力量的经典动作，主要锻炼胸大肌、三角肌前束和肱三头肌。

## 动作要领

### 准备姿势
1. 仰卧在卧推凳上，眼睛位于杠铃正下方
2. 双脚踩实地面，臀部贴紧凳面
3. 收紧肩胛骨，挺胸沉肩
4. 双手握杠，握距约为肩宽的1.5倍

### 下放阶段
1. 控制杠铃缓慢下放至胸部
2. 肘部与躯干呈45度角，不要完全外展
3. 保持手腕中立，不要过度后伸
4. 触胸点在乳头线上方

### 推举阶段
1. 胸部和三头肌发力，将杠铃推起
2. 保持杠铃轨迹稳定，不要晃动
3. 手臂伸直但不过伸，锁定肘关节

## 常见错误

1. **臀部离凳**：会减少胸肌参与，增加受伤风险。应保持臀部贴紧凳面。

2. **肘部过度外展**：容易导致肩关节受伤。应保持肘部与躯干呈45度角。

3. **过度反弓腰部**：可能导致下背部受伤。应保持核心稳定，腰椎自然曲度。

4. **握距过宽或过窄**：过宽增加肩部压力，过窄主要锻炼三头肌。

## 训练建议

- 初学者建议从空杆开始练习
- 每周训练1-2次，每次3-4组，每组6-10次
- 必须有保护人员，或在安全架内训练

## 变式动作

- 上斜卧推：侧重上胸
- 下斜卧推：侧重下胸
- 窄距卧推：侧重三头肌
- 哑铃卧推：活动范围更大
