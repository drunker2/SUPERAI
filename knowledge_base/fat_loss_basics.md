# 减脂训练计划指南

减脂的核心原则是创造热量缺口，即消耗的热量大于摄入的热量。合理的训练计划能帮助你高效减脂。

## 训练原则

### 1. 力量训练优先
力量训练能保持甚至增加肌肉量，提高基础代谢率。减脂期间建议：
- 每周3-4次力量训练
- 以复合动作为主（深蹲、硬拉、卧推等）
- 重量适中，每组10-15次

### 2. 有氧训练配合
有氧运动是消耗热量的有效方式：
- 低强度稳态有氧（LISS）：心率60-70%，30-45分钟
- 高强度间歇训练（HIIT）：心率80-90%，15-25分钟
- 建议每周3-5次有氧训练

### 3. 训练顺序
推荐先力量训练后有氧训练：
- 力量训练消耗糖原
- 后续有氧更多消耗脂肪

## 周计划示例

**周一**：力量训练（上肢）+ 20分钟有氧
**周二**：力量训练（下肢）+ 20分钟有氧
**周三**：休息或轻度有氧
**周四**：力量训练（全身）+ HIIT
**周五**：有氧45分钟
**周六**：休息
**周日**：户外活动或休息

## 饮食建议

减脂期热量摄入建议：
- 男性：低于日常消耗300-500卡
- 女性：低于日常消耗200-300卡

营养配比：
- 蛋白质：每公斤体重1.6-2.2克
- 碳水化合物：占总热量40-50%
- 脂肪：占总热量25-35%

## 注意事项

1. 不要过度节食，会导致代谢下降
2. 保证充足睡眠，每晚7-8小时
3. 多喝水，每天至少2升
4. 循序渐进，每周减重0.5-1公斤为宜
