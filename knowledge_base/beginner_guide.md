# 健身新手入门指南

刚开始健身？这份指南帮助你正确起步。

## 第一步：明确目标

常见目标：
- 减脂：降低体脂率
- 增肌：增加肌肉量
- 增力：提高力量水平
- 健康：改善身体素质

## 第二步：制定计划

### 训练频率
- 新手建议每周3次
- 全身训练为主
- 休息日穿插

### 训练内容
每次训练包括：
- 热身：10分钟
- 力量训练：40-60分钟
- 有氧（可选）：20分钟
- 拉伸：10分钟

### 基础动作
1. 深蹲
2. 硬拉
3. 卧推
4. 划船
5. 推举

## 第三步：注意饮食

### 基本原则
- 蛋白质每公斤体重1.5克
- 碳水化合物适量
- 蔬菜水果充足
- 水分每天2升以上

### 避免的食品
- 加工食品
- 含糖饮料
- 油炸食品
- 过量酒精

## 第四步：循序渐进

### 新手常见错误
1. 训练量过大
2. 忽视休息
3. 急于求成
4. 忽视动作标准

### 建议进度
- 第1-4周：学习动作
- 第5-8周：建立习惯
- 第9-12周：增加强度

## 保持动力

1. 设定小目标
2. 记录训练日志
3. 找训练伙伴
4. 享受过程
