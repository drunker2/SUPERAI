# 肩关节健康指南

肩关节是人体最灵活的关节，也是最容易受伤的关节之一。

## 肩关节结构

肩关节由：
- 肱骨头
- 肩胛骨关节盂
- 锁骨
- 周围肌肉和韧带

组成，活动范围大但稳定性相对较低。

## 常见损伤

### 肩袖损伤
- 原因：过度使用、动作不当
- 症状：抬臂疼痛、夜间疼痛
- 预防：加强肩袖训练

### 肩峰撞击
- 原因：肩峰下空间不足
- 症状：举手时疼痛
- 预防：改善姿势、加强肩袖

### 肩关节脱位
- 原因：外力或极端动作
- 需就医处理

## 预防训练

### 肩袖训练
1. 弹力带外旋
2. 弹力带内旋
3. 面拉
4. 肩胛骨俯卧撑

### 姿势改善
1. 胸肌拉伸
2. 背部加强
3. 避免含胸驼背

## 训练建议

1. 热身充分
2. 控制重量，避免借力
3. 保持肩胛骨稳定
4. 听从身体信号，疼痛时停止

## 出现疼痛时

1. 停止引起疼痛的动作
2. 冰敷缓解炎症
3. 必要时就医
4. 恢复后逐步训练
