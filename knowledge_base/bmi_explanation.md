# BMI指数解析

BMI(Body Mass Index)身体质量指数是评估体重是否健康的常用指标。

## 计算公式

BMI = 体重(kg) / 身高²(m²)

例如：身高1.75m，体重70kg
BMI = 70 / (1.75)² = 22.86

## 分类标准（亚洲人）

| BMI范围 | 分类 |
|---------|------|
| < 18.5 | 偏瘦 |
| 18.5-23.9 | 正常 |
| 24-27.9 | 超重 |
| ≥ 28 | 肥胖 |

## BMI的局限性

### 不适用的群体
1. 运动员：肌肉量大，BMI可能偏高
2. 老年人：肌肉流失，BMI可能正常但体脂高
3. 儿童和青少年：需使用年龄别BMI

### 局限性
1. 不区分脂肪和肌肉
2. 不反映体脂分布
3. 不考虑骨骼密度

## 更全面的评估

建议结合以下指标：
1. **腰围**：男性<90cm，女性<80cm
2. **体脂率**：男性15-20%，女性20-25%
3. **腰臀比**：男性<0.9，女性<0.8

## 健康建议

### BMI过低
- 增加热量摄入
- 力量训练增加肌肉
- 关注营养均衡

### BMI过高
- 创造热量缺口
- 增加有氧运动
- 控制饮食热量
